## git cheat sheet

### branching
```
git checkout master
git pull
git branch <name of branch>
git checkout <name of branch>
```

### rebasing
```
# stash or commit changes
git checkout master
git pull
git checkout <name of branch>
git rebase master
```

### rebasing merge conflicts
```
# fix conflicts
git add <files>
# if more
git rebase --continue
```
